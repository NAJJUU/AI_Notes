1. 소개
    1) 구글이 만든 머신러닝을 위한 엔드투엔드 오픈소스 플랫폼
        -기계학습 프레임워크
    2) GPU 연산의 CUDA(Computed Unified Device Architecture)를 지원함으로써 빠른 처리 가능
        -GPU 연동을 통해 효율적으로 딥러닝 모델을 학습할 수 있음
    3) 2.x 버전
        -CPU, GPU 통합 플랫폼으로 제공
        -프로그램 소스 차원에서는 변경 없이 GPU 사용 가능
    4) Tensor라는 다차원 데이터 구조 사용
        -텐서를 Numpy 배열처럼 사용할 수 있음

2. GPU 사용 여부 체크

3. 텐서 소개 및 생성 
    1) TensorFlow에서의 텐서(tensor)는 기능적으로 Numpy와 매우 유사함
    2) 기본적으로 다차원 배열을 처리하기에 적합한 자료구조
    3) TensorFlow에서의 텐서(tensor)는 "자동미분" 기능 제공

4. 텐서의 속성
    1) 기본 속성
        -모양(shape)
        -자료형(data type)
        -저장된 장치

5. 텐서 초기화
    -다른 텐서로부터 텐서 초기화 

6. 텐서의 형변환 및 차원 조작
    1) 텐서는 넘파이(Numpy) 배열처럼 조작할 수 있음
    2) 텐서의 특정 차원 접근
